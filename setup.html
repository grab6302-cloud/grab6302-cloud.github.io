📖 Hướng Dẫn Setup Tool Dò

Chỉ dành cho máy tính (Laptop/PC) để cấp nguồn đáp án cho toàn server.

Bước 1: Tải và cài đặt Node.js Vào trang chủ nodejs.org tải bản LTS mới nhất và cài đặt (Cứ bấm Next liên tục là xong).
Bước 2: Tải file chạy Tool Admin sẽ gửi bạn 1 file tên là proxy.js. Hãy lưu file này vào máy tính của bạn (Ví dụ: Desktop).
Bước 3: Mở Terminal / CMD Mở thư mục chứa file proxy.js -> Nhấp chuột phải vào khoảng trống -> Chọn Open in Terminal (Hoặc gõ cmd lên thanh địa chỉ thư mục rồi Enter).
Bước 4: Chạy Server Tool Dò Trên cửa sổ Terminal đen ngòm, hãy nhập lệnh sau và bấm Enter: node proxy.js Nếu thấy chữ "Server ready!" và "Port 9876" hiện ra là bạn đã cắm tool thành công. Đừng tắt cửa sổ đen này khi đang kiếm đáp án nhé!
